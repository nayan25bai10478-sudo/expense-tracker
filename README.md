# 💰 Expense Tracker (Python)

## 📌 Project Overview

The Expense Tracker is a simple GUI-based application developed using Python and Tkinter. It helps students manage their daily expenses efficiently by recording, storing, and displaying spending data.

---

## ❗ Problem Statement

Students often struggle to keep track of their expenses, leading to poor financial management and overspending.

---

## 🚀 Features

* Add new expenses (description, amount, category)
* View all recorded expenses
* Automatic total calculation
* Data stored using JSON file (persistent storage)
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* JSON (Data storage)

---

## ▶️ How to Run the Project

1. Install Python (if not already installed)
2. Download or clone this repository
3. Open terminal in project folder
4. Run the following command:

   python expense_tracker.py
